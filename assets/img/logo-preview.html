Astuce - Convertir SVG en PNG/JPG pour Facebook & Instagram

Les SVG sont parfaits pour le web. Mais Facebook et Instagram préférent PNG ou JPG. Pour convertir :

  1. Ouvrez le SVG dans Chrome/Edge
  2. Faites une capture d'écran (Outil capture Windows : Win + Maj + S)
  3. OU utilisez un convertisseur gratuit : cloudconvert.com/svg-to-png
